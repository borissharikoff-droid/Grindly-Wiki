📊 Real-Time Activity Tracking

Grindly monitors your active window and categorizes every minute you spend on your PC. Coding in VS Code? That's Developer XP. Browsing Figma? Designer XP. Playing games? Gamer XP. Everything is tracked automatically — no manual input required.

  • Automatic window detection via Win32 APIs
  • Per-app and per-category time breakdowns
  • Daily, weekly and all-time analytics
  • Idle detection — AFK time doesn't count
  • 100% local — your data stays on your machine

XP Formula

Each skill has 99 levels. XP required for level L:

xpForLevel(L) = floor(pow(L/99, 2.2) * 3,600,000)

Max XP (level 99) = 3,600,000. See Skills → XP Formula for full table.
