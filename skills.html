Meta

Levelling Curve

  • Each skill has 99 levels, designed to take roughly ~1,000 hours to max.
  • 1 XP = 1 second of tracked activity in the mapped category.
  • The curve has 3 segments for smooth early-game progression:

Three-Segment Curve

  • Levels 1–10 (Fast Start): Triangular ramp — XP = 120 × L × (L+1) / 2
    L1 costs only 2 min, L10 costs ~1.8h total. Designed for quick early dopamine.
  • Levels 11–25 (Transition): Smooth cubic interpolation between L10 (6,600 XP) and L25 (126,230 XP).
    Uses exponent 1.3 for a gradual difficulty ramp.
  • Levels 26–99 (Exponential): XP = floor((L/99)^2.2 × 3,600,000)
    Full power-law curve. Each level costs progressively more time.

Useful Numbers

  • Max XP per skill: 3,600,000 (1,000 hours in seconds)
  • Total skills: 12 — max total level: 1,188
  • XP per second: 1 (before multipliers)
  • Time to reach level 10: ~1.8 hours (fast early progression)
  • Time to reach level 50: ~153 hours (mid-game)
  • Time to max (level 99): ~1,000 hours (end-game grind)

How Skill XP Is Earned

  • Passive skills (Developer, Designer, etc.) gain XP automatically while the tracked app is in focus.
  • 1 real second = 1 XP in the mapped skill. No manual input needed.
  • If you switch between apps, each skill gets XP proportional to time spent.
  • Idle time (no input for extended period) does not grant XP.
  • Uncategorized apps fall back to Researcher skill.

In-Game Skills
These don't track apps — they use in-game actions:

Farmer — XP from planting seeds and harvesting crops

Warrior — XP from defeating arena bosses

Crafter — XP from completing crafts

Grindly — XP from time spent in the Grindly app itself
XP Multipliers Stack
Base XP can be boosted by:

1. Equipped loot perks (skill-specific or global XP boosts)

2. Grindly skill bonuses (up to +43% all skill XP at LVL 99)

All multipliers stack multiplicatively.

XP Multiplier Stacking

All XP multipliers stack multiplicatively. Here are all sources:

  • Grindly Skill (max LVL 99) up to ×1.43 (+43%)
  • Equipped item: Skill-specific XP boost varies per item
  • Equipped item: Global XP boost +5% to +8% per item

Example: Maxed Grindly + Equipped Gear

With Grindly LVL 99 (×1.43) + a Void Helmet (×1.08) + a Void Ring (×1.08):
1 × 1.43 × 1.08 × 1.08 = 1.667×
That's 66.7% bonus XP per second, cutting the time-to-max to ~600 hours.

Craft Speed Stacking

Crafter level speed and Grindly craft speed multiply together:

  • Crafter Skill (max LVL 80) ×0.40 (−60%)
  • Grindly Skill (max LVL 99) ×0.75 (−25%)
  • Combined Maximum ×0.30 (−70%)
